## AI 도입의 시작은, 
사내 데이터를 이해하는 AI여야 합니다 

- 
### 우리 회사는, 
우리 데이터로 움직입니다 

범용 AI는 회사의 문서와 맥락을 모릅니다. 사내 데이터를 이해하는 AI가 진짜 업무 파트너입니다 

- 
### 정확한 답은, 
내부에서 나옵니다 

외부 정보 기반 답변은 일반적입니다. 사내 문서를 기반으로 해야 실무에서 실제로 활용할 수 있습니다 

- 
### 보안 없는 AI는 
사용할 수 없습니다 

외부 AI에 사내 데이터를 입력하면 보안 리스크가 발생합니다. 완전한 데이터 격리가 필요합니다 

- 
### 검색이 아니라, 
업무가 되어야 합니다 

단순 정보 제공이 아닌 결과를 도출하는 도구여야 합니다. AI는 업무의 도구입니다 

- 
### 전 직원이 써야, 
AI 도입의 의미가 있습니다 

쉽고 빠른 도입으로 전사 확산이 이루어져야 AI 투자의 가치가 실현됩니다 

- 
### AI는 도입보다 
운영이 중요합니다 

실제 업무 흐름에 자연스럽게 통합되어야 지속적인 가치를 만들어냅니다

달라야 합니다 

오피스에이전트는 범용 모델을 그대로 사용하는 AI가 아닙니다. 기업 환경에 맞게 최적화된 고성능 오픈소스 LLM을 기반으로 동작합니다. 사내 문서와 데이터 구조를 반영해 컨텍스트를 이해하고, 업무 맥락에 맞는 정보를 정확하게 찾아 답변을 생성합니다.
